# Scripts Directory

This directory contains utility scripts for the MCP Blink Memory project.

## Available Scripts

- `build.sh` - Build the TypeScript project
- `dev.sh` - Start development server with hot reload
- `test.sh` - Run test suites
- `clean.sh` - Clean build artifacts and temporary files
- `setup.sh` - Initial project setup
- `deploy.sh` - Deployment script

## Usage

```bash
# Make scripts executable (Linux/macOS)
chmod +x scripts/*.sh

# Run a script
./scripts/build.sh
```

## Windows Users

Use `.bat` versions of scripts or run through Git Bash/WSL.
